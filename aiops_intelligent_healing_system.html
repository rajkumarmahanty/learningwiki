📄 Component Breakdown

  • langchain_agent.py: The core reasoning engine that generates Kubernetes JSON patches.
  • state_manager.py: Uses Redis to ensure Slack approval tokens are secure and temporary.
  • slack_handler.py: Handles interactive "Block Kit" messages for human-in-the-loop validation.
  • upsert_rag_data.py: The "Teacher" script—converts text runbooks into searchable vectors.
  • k8s_tools.py: Wrapper for the Kubernetes Python SDK to execute Strategic Merge Patches.

🛠️ Step-by-Step Execution Guide

Phase 1: Environment Boot

Ensure your .env file is populated, then launch the local stack:

docker-compose up --build -d

Phase 2: Seeding Intelligence

Feed your local agent the engineering expertise stored in your runbooks:

python scripts/upsert_rag_data.py

Phase 3: Import n8n Workflow

  1. Open http://localhost:5678.
  2. Import the JSON from n8n_workflows/.
  3. Activate the workflow to start listening for Prometheus webhooks.

Phase 4: The Fire Drill

Execute the simulator to trigger a predictive alert and receive a Slack notification:

python scripts/test_webhook.py
